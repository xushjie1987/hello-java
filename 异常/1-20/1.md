# 把异常对象转换成String

`注意：`

```
1. 思路是把原本打印到java.io.PrintWriter中的数据重定向到java.io.StringWriter；
2. 来自flink-core制品的org.apache.flink.util.ExceptionUtils；
```

`代码：`

```xml
        <dependency>
            <groupId>org.apache.flink</groupId>
            <artifactId>flink-core</artifactId>
            <version>1.6.2</version>
        </dependency>
```

```java
    public static void main(String[] args) {
        System.out.println(ExceptionUtils.stringifyException(new NullPointerException("Test")));
    }
```
